# Tenzies Game

Practice project as part of the freeCodeCamp course - [React JavaScript Library 2022](https://www.youtube.com/watch?v=bMknfKXIFA8)


## Tech Used

- JavaScript
- ReactJS
- HTML
- CSS
- [React-confetti effect](https://github.com/alampros/react-confetti)
